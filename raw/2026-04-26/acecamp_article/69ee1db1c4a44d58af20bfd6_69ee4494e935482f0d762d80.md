# 一、Bedrock 收入

# 市场共识对比

卖方研报（BofA、GS、UBS等）对AWS AI收入的讨论集中在全球层面：CEO Jassy披露AI收入 run-rate 达\$15B（约占 AWS 总收入 10%），Bedrock 在 3 月 token 处理量环比翻倍。但从未有卖方拆分过中国区 Bedrock 的具体收入数字。  
市场对 AWS AI 增速的共识是全球层面 25-30% YoY 的 AWS 整体增速，AI 部分 triple-digit 增长。

# 专家增量

中国区Q1 Bedrock收入约1.5亿美元，同比增速约150%，预计2026年底中国区增速达300%。  
中国AI收入占全球总盘仅1%-3%。  
中国区 Q1 AI 云总收入约 250 亿元（含 Inf 级 150 亿、Bedrock standalone 150 亿等细分）。

# 二、Bedrock 行业分布

# 专家增量

全球Bedrock行业占比中，加密货币板块约30%，AI原生应用（含DNB）约40%（头部企业如字节、腾讯占 80%），FSI、ENT、healthcare 等占比较小。  
FSI、DNB行业受中东数据中心爆炸影响，增长不及往年。

# 市场共识对比

卖方研报几乎从未披露过Bedrock的行业客户分布。市场共识中，AWS AI客户的讨论集中在Anthropic（占 Bedrock 80%-90%的 API 调用量）、企业级客户（Capital One 等传统大客户），以及 AInative 公司。  
加密货币占Bedrock收入30%这一数据点，在公开研报和公告中完全没有出现过，属于纯增量信息。  
中东数据中心爆炸对FSI/DNB增速的影响，也是市场未曾讨论的事件冲击。

# 增量价值

这彻底改变了对Bedrock客户结构的理解——市场默认Bedrock主要服务AI模型公司和传统企业，但实际上加密货币是第一大垂直行业。这意味着Bedrock收入的波动性可能高于预期（受加密周期影响），也暗示AI在加密/Web3领域的渗透速度远超市场感知。

# 三、Anthropic分成比例与AWS的AI收入构成

# 专家增量

AWS 与 Anthropic 的分成比例为 AWS 分 30%，Anthropic 的 API 调用量和收入占 Bedrock 的80%-90%。

# 市场共识对比

市场对 Anthropic 与 AWS 的分成模式一直缺乏共识。浙商计算机纪要提到”Anthropic 与 AWS 分成毛利润的 50%“，Platformer 报道称 Anthropic 采用 gross revenue 报告方式（包含分给云伙伴的部分），UBS 深度研报讨论了 Anthropic 在 AWS vs GCP 的支出分配但未明确分成比例。  
“AWS 分 30%”这一具体数字是首次从 AWS 内部视角披露，且与浙商纪要的”50%毛利润分成”口径不同（一个是收入分成，一个是毛利润分成），提供了新的交叉验证维度。

# 增量价值

这一数据对估算 AWS 从 Anthropic 获得的实际收入至关重要。以 Anthropic 当前\$30B ARR 计算，若 80-90%通过 Bedrock，AWS 从中获得的分成收入约\$7.2-8.1B，这一量级与卖方对 AWS AI 收入\$15B run-rate的估算高度吻合，说明Anthropic贡献了AWS AI收入的约一半。

# 四、Agent 算力乘数效应

# 专家增量

Agent对算力的拉动具有乘数效应：一次复杂操作触发20次以上模型调用，token数为单轮的5-10倍，对GPU算力消耗是传统生成AI（如ChatGPT对话式应用）的3-10倍。  
当前 agent 带来的收入占比约 1/3。  
未来将向API调用+算力租赁+存储+数据处理的复合收入转变。

# 市场共识对比

市场共识认可Agent是AI下一阶段的关键方向，UBS调查显示企业正加速采用Bedrock等AI开发平台，GS研报提到”AI Goes Agentic”。但对Agent具体带来多少算力增量，市场缺乏量化框架。

# 五、Trainium 3出货量与下一代芯片路线图

# 专家增量

Trainium 3：2026 年出货 50 万颗，主要供 Bedrock 内部及 Anthropic、Uber 等战略合作伙伴。  
训练成本较GPU低50%，性价比高30%-40%。  
下一代芯片（Trainium 4）预计 2027 年量产，采用 3 纳米工艺，聚焦高算力密度与低功耗。  
目标 2030 年自研 AI 算力占比超 50%。

# 市场共识对比

市场共识：AWS CEO 在股东信中称芯片业务 run-rate 超\$20B，Trainium2 已售罄，Trainium3 几乎全部被预订，Trainium4 需求强劲。UBS 深度研报指出 Trainium2 性能弱于 Nvidia GPU，市场对Trainium 押注”过早”存疑。GS 研报提到 AWS 已部署超 100 万颗 Trainium 芯片。  
但 “2026年出货50万颗Trainium 3”这一具体数字，以及 “下一代采用3纳米、2027年量产”的路线图，在公开研报中未见披露。  
市场共识中，AWS 的自研芯片目标是”50%+ compute mix on Trainium 2 and Trainium 3”

# 增量价值

50万颗的出货量为评估Trainium 3对AWS收入的贡献提供了基础。按UBS模型中Trainium芯片\$2.24/小时的list price（60%折扣后）估算，50万颗Trainium 3 全年可贡献数十亿美元收入。3纳米工艺的路线图也暗示AWS在芯片设计上的投入正在加速追赶。

# 六、推理场景芯片分布

# 专家增量

推理场景中，Inferentia 芯片占 30%-40%，NVIDIA GPU 占 60%-70%。  
复杂大模型推理以GPU为主，轻量级模型优先用Inferentia。   
目标1-2年内将Inferentia在推理场景占比提升至50%以上。

# 市场共识对比

• AWS CEO 曾表示”Trainium powers more than 50% of inference that goes through Bedrock”，但这指的是Bedrock平台内部，而非AWS整体推理场景。  
UBS调查显示Nvidia仍是LLM训练和推理的首选平台，AWS芯片排名第三。市场共识认为Inferentia”barely mentioned by Amazon anymore”。

# 增量价值

这一数据表明AWS自研芯片在推理端的渗透已相当可观（30-40%），远高于市场”Inferentia几乎不被提及”的悲观印象。若目标实现，将显著降低AWS对Nvidia的依赖和成本压力。

# 七、GPU涨价细节

# 专家增量

2026 年初对新签/续约客户涨价，仅涉及 P5、G5 等高端 GPU 实例，成本增加 20%。  
老客户合约期内价格不变。  
部分客户转向 Spot 实例、自研芯片或混合部署。

# 市场共识对比

广发证券纪要提到”AWS 2026年1月对部分搭载8颗H200 GPU的实例提价约15%“，本纪要给出的20%幅度略高于此前报道的15%，且明确了”仅新签/续约”的范围限制。  
BofA 研报提到”core pricing”受益于 capacity shortages，但未给出具体涨价幅度。

# 八、AI 云毛利率

# 专家增量

AI业务毛利率约40%，包含API调用、IaaS、RAG API包及所有AI组件。  
对整体 margin 影响不大。

# 市场共识对比

BofA 研报估算”if we assume a 15% operating margin for AI, the AI contribution could be impacting overall AWS margins by approx. 2ppts”，暗示 AI operating margin 约 15%。   
UBS/GS 研报讨论 AWS 整体 OI margin 在 30-35%区间，AI mix 上升是 margin 压力来源。  
“40%毛利率”这一数字是首次从AWS内部视角给出的AI业务盈利水平。注意这是毛利率而非营业利润率，与 BofA 假设的 15% operating margin 并不矛盾（毛利率 40%扣除研发、销售等费用后，operating margin 可能在 15-20%区间）。

40%的毛利率低于传统云业务（估计60%+），说明即便对于亚马逊这种业务偏dirty且heavy来说，AI收入占比的提升也是拉低整体利润率的。这体现了美股巨头生意模式恶化的窘境。但毛利率这块并非不可接受，且随着规模效应和自研芯片渗透率提升，有改善空间。
